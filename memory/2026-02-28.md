### 技术细节
**Modal 拦截问题：**
- Ant Design Modal 使用 `.ant-modal-wrap` 和 `.ant-modal-mask` 作为遮罩
- 遮罩层会拦截所有点击事件
- 解决方案：
  1. 使用 `page.evaluate()` 直接操作 DOM 隐藏遮罩
  2. 使用 `force: true` 强制点击（但可能不生效）
  3. 使用键盘快捷键（Escape）关闭 Modal

**API 认证问题：**
- AgileConfig 使用 JWT Token 或 Cookie 认证
- Token 可能存储在：
  - localStorage（key: `token` / `Authorization`）
  - Cookie（name: `.AspNetCore.Identity.Application` 或类似）
  - 请求头（Header: `Authorization: Bearer xxx`）
- 需要从浏览器获取完整的认证信息
